# github-username-email
Script to fetch emails of the given username

## Configuration

1. Install the required dependencies.
   ``` pip3 install -r requirements.txt ```
2. Make a .env file for github key configuration.
   ``` GH_SAML_TOKEN = <your_github_token>

## Usage

``` python3 getEmailbyUsername.py <github-username> ```

or run without args to be prompted for input

``` python3 getEmailbyUsername.py ```
